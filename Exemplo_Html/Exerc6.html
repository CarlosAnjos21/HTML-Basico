Descrição do curso

Este curso apresenta conceitos básicos de web e mostra as partes e funções das tags em um documento HTML. Ensina, também, a formatar uma página, inserir imagens e muito mais! Tudo isso, com explicações simples e de fácil compreensão.

Estrutura do curso

  1. Modulo 1 Introdução
  2. Modulo 2 HTML e a web
  3. Modulo 3 Criando um documento HTML
  4. Modulo 4 Formatando um documento HTML
  5. Modulo 5 Criando listas
  6. Modulo 6 Inserindo imagens
  7. Modulo 7 Criando links
  8. Modulo 8 Combinando tags
  9. Modulo 9 Usando imagens mapeadas
